## 🧠 Student Stress Prediction and Monitoring System

This project focuses on developing an intelligent system that predicts and monitors student stress levels based on daily lifestyle factors such as study hours, sleep patterns, physical activity, social interaction, and academic performance.

The system uses Machine Learning techniques to estimate stress levels and provides personalized recommendations to help students maintain a healthy balance. It also tracks daily stress data, analyzes trends over time, and offers insights such as weekly averages and stress improvement percentages.

Additionally, the application includes user authentication, daily tracking restrictions (one entry per day), graphical visualization of stress trends, and an admin panel for institutional-level analysis.

### 🎯 Objective

To help students understand, monitor, and manage their stress levels through data-driven insights and actionable recommendations.

### 🚀 Key Features

* Machine Learning-based stress prediction
* Daily stress tracking (one entry per day)
* Personalized recommendations
* Weekly average and trend analysis
* Stress improvement percentage tracking
* Interactive graph visualization
* User authentication system
* Admin dashboard for overall analytics

### 🧩 Technologies Used

* Python
* Flask (Web Framework)
* Scikit-learn (Machine Learning)
* SQLite (Database)
* HTML, CSS, JavaScript (Frontend)
* Chart.js (Graph Visualization)

### 📊 Outcome

A complete end-to-end Data Science web application that not only predicts stress but also provides meaningful insights and supports better mental well-being through continuous monitoring.
